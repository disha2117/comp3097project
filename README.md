# SmartToDo iOS Application

## 📌 Project Description
SmartToDo is an iOS application developed using Swift and UIKit.  
The app allows users to create, view, and manage tasks efficiently with a simple and user-friendly interface.

---

## 🚀 Features
- Add new tasks
- View list of tasks
- View task details
- Mark tasks as completed
- Navigation between multiple screens
- Clean and simple UI

---

## 🛠️ Technologies Used
- Swift
- UIKit
- Xcode
- Storyboard (Auto Layout)

---

## 📂 Project Structure
- `ViewController.swift` → Main task list screen
- `AddTaskViewController.swift` → Add new task
- `TaskDetailsViewController.swift` → Task details screen
- `Task.swift` → Data model
- `Main.storyboard` → UI design and navigation

---

## 🔄 Functionality Implemented
- Navigation between screens using Segues
- Task data structure using arrays
- TableView for displaying tasks
- Basic data handling (read/write in memory)
